Este repositório pode ser instalado de duas formas: clonando diretamente do GitHub
ou instalando como pacote npm. Ambas as opções permitem que você tenha uma base
de conhecimento pronta para ser consumida por agentes de IA.

## Instalação via GitHub

::steps
### Clonar o repositório

Clone o repositório do GitHub para sua máquina local:

```bash [Terminal]
git clone https://github.com/anilsonlopes/context-directory.git
```

Ou usando SSH:

```bash [Terminal]
git clone git@github.com:anilsonlopes/context-directory.git
```

### Navegar para o diretório

Entre no diretório do projeto:

```bash [Terminal]
cd context-directory
```

### Instalar dependências

Instale as dependências do projeto:

```bash [Terminal]
npm install
```

Ou usando outros gerenciadores de pacotes:

```bash [Terminal]
# Usando pnpm
pnpm install

# Usando yarn
yarn install

# Usando bun
bun install
```

### Iniciar servidor de desenvolvimento

Inicie o servidor de desenvolvimento para visualizar a documentação:

```bash [Terminal]
npm run dev
```

A documentação estará disponível em <http://localhost:3000>

### Personalizar o conteúdo

Edite os arquivos Markdown no diretório `content/` para personalizar a base
de conhecimento conforme suas necessidades.
::

## Instalação via npm

::steps
### Instalar o pacote

Instale o pacote `@anilsonlopes/context-directory` via npm:

```bash [Terminal]
npm install @anilsonlopes/context-directory
```

Ou usando outros gerenciadores:

```bash [Terminal]
# Usando pnpm
pnpm add @anilsonlopes/context-directory

# Usando yarn
yarn add @anilsonlopes/context-directory

# Usando bun
bun add @anilsonlopes/context-directory
```

### Usar em seu projeto

Após a instalação, você pode importar e usar o conteúdo do repositório em seu
projeto. O conteúdo estará disponível em `node_modules/@anilsonlopes/context-directory/content/`.

### Copiar conteúdo para seu projeto

Se preferir ter o conteúdo diretamente em seu projeto, copie os arquivos:

```bash [Terminal]
cp -r node_modules/@anilsonlopes/context-directory/content ./seu-diretorio-content
```

### Iniciar servidor de desenvolvimento

Se você copiou a estrutura completa do repositório, inicie o servidor:

```bash [Terminal]
npm run dev
```
::
